# How to use

<p align="center">
    Just simplely enter your color code in HEX, RGB, HSL, or CMYK format into the input field at the middle of the page.
</p>

# Accepted Color Formats

| Color Code                          | Description                                                                                                                                                                                                                                                                                     | Example                                          |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| HEX (Hexadecimal)                   | <ul><li>HEX that can without '#'. 3 or 6 characters of '0-9' and 'A-F'.</li></ul>                                                                                                                                                                                                               | `#1A2B3C`, `#1AF`, `1A2B3C`, `1AF`               |
| RGB (Red, Green, Blue)              | <ul><li>RGB that can without 'rgb' and brackets.</li><li>Three sets of 3 positive digits, each separated by a comma.</li><li>Values over 255 will be capped at 255.</li></ul>                                                                                                                   | `rgb(255, 128, 0)`, `999,999,999`                |
| HSL (Hue, Saturation, Lightness)    | <ul><li>HSL that can without 'hsl' and brackets.</li><li>Three positive digits for hue.</li><li>two sets of three positive digits followed by a percent sign for saturation and lightness.</li><li>Values over 360 for hue and over 100% for saturation and lightness will be capped.</li></ul> | `hsl(180, 50%, 75%)`, `999,999%,999%`            |
| CMYK (Cyan, Magenta, Yellow, Black) | <ul><li>CMYK that can without 'cmyk' and brackets.</li><li>Four sets of three positive digits followed by a percent sign.</li><li>Values over 100% will be capped.</li></ul>                                                                                                                    | `cmyk(0%, 50%, 100%, 0%)`, `999%,999%,999%,999%` |
